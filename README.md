# 技术支撑事业部项目管理平台 V1.1

基于 `PHP + MySQL` 的轻量级项目管理系统，面向技术支撑事业部的项目跟进、人员绩效、出勤管理、项目文件管理与统计报表场景。

## 功能概览

- 项目管理
  - 新增、编辑、预览、分页展示
  - 按销售、反馈标签、月份、支撑人员、工单标签筛选
  - 批量勾选与批量操作
  - Excel 导入、导出
- 人员绩效
  - 按年、月、周统计绩效
  - 查看个人每日项目任务与绩效明细
  - 一键生成个人绩效表与整月绩效汇总
- 出勤管理
  - 自动识别项目出勤
  - 管理员可手动调整为空闲、调休等状态
- 项目文件
  - 文件上传、下载、预览
  - 关键字搜索
- 统计报表
  - 项目总数、总工时、总天数等概览
  - 项目导出预览与报表导出
- 权限控制
  - `admin` 管理员
  - `editor` 项目编辑员
  - `auditor` 审核员

## 运行环境

- PHP 7.3+
- MySQL 5.7+
- Apache / Nginx / phpStudy

## 本地运行

1. 将项目放入站点目录，例如 `F:\phpstudy_pro\WWW\pre`
2. Web 根目录指向项目根目录
3. 按实际环境修改 [config/app.php](F:\phpstudy_pro\WWW\pre\config\app.php) 中的数据库配置
4. 访问 `index.php` 进入系统

示例数据库配置：

```php
'db' => [
    'host' => '127.0.0.1',
    'port' => 3306,
    'database' => 'pre',
    'username' => 'root',
    'password' => 'root',
    'charset' => 'utf8mb4',
    'auto_create_database' => false,
],
```

## Excel 说明

### 项目导入

- 支持 `.xlsx`、`.csv`
- 导入字段已与当前导出模板对齐
- 支持将 Excel 内容直接导入项目平台

### 项目导出

- 支持导出项目计划表
- 已包含反馈标签列
- 已将“完成评价【销售反馈简述】”调整到对应导出列

### 绩效导出

- 支持一键生成整月绩效汇总
- 支持一键生成个人绩效明细

## 权限说明

- 管理员：可管理项目、人员绩效、出勤、文件、报表、系统设置
- 项目编辑员：可查看项目、出勤，可新增项目，但不能修改既有项目和人员绩效
- 审核员：可查看并处理项目相关管理内容
