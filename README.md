# 技术支撑事业部项目管理平台 V1.2

基于 `PHP + MySQL` 的轻量级项目管理系统，面向技术支撑事业部的项目跟进、人员绩效、出勤管理、项目文件管理与统计报表场景。

## 功能概览

- 项目管理
  - 新增、编辑、预览、分页展示
  - 按销售、反馈标签、月份、支撑人员、工单标签筛选
  - 批量勾选与批量操作
  - Excel 导入、导出
- 人员绩效
  - 按年、月、周统计绩效
  - 查看个人每日项目任务与绩效明细
  - 一键生成个人绩效表与整月绩效汇总
- 出勤管理
  - 自动识别项目出勤
  - 管理员可手动调整状态
- 项目文件
  - 文件上传、下载、预览
  - 关键字搜索
- 统计报表
  - 项目总数、总工时、总天数概览
  - 图表联动
  - 项目导出预览
  - Excel 导出
- 账号权限
  - `admin` 系统管理员
  - `editor` 普通用户

## 运行环境

- PHP 7.3+
- MySQL 5.7+
- Apache / Nginx / phpStudy

## 安装部署

部署后先访问：

```text
/install.php
```

安装页支持：

- 测试数据库连接
- 检查数据库是否存在
- 按需自动创建数据库
- 初始化表结构
- 初始化基础系统设置
- 创建首个系统管理员账号
- 自动写入 `config/app.php`

说明：

- 安装流程不会写入测试项目数据
- 首次安装由你自己配置系统管理员账号和密码
- 系统安装完成后再访问 `index.php`

## 本地运行

1. 将项目放入站点目录，例如 `F:\phpstudy_pro\WWW\pre`
2. Web 根目录指向项目根目录
3. 访问 `install.php` 完成数据库初始化
4. 安装完成后访问 `index.php`

## Excel 说明

### 项目导入

- 支持 `.xlsx`、`.csv`
- 导入字段已与当前导出模板对齐
- 支持将 Excel 内容直接导入平台

### 项目导出

- 支持导出项目计划表
- 已包含反馈标签列
- 已将“完成评价【销售反馈简述】”放到对应导出列

### 绩效导出

- 支持一键生成整月绩效汇总
- 支持一键生成个人绩效明细

## 权限说明

- 系统管理员：可管理项目、人员绩效、出勤、文件、报表、系统设置，并管理账号
- 普通用户：可查看项目、出勤，可新增项目，但不能修改既有项目和人员绩效

## 账号管理
