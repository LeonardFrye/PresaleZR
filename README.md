# 技术支撑事业部项目管理平台 V1.3

基于 `PHP + MySQL` 的轻量级项目管理系统，面向技术支撑事业部的项目跟进、人员绩效、出勤管理、项目文件与统计报表场景。

## 功能概览

- 项目管理
  - 支持新增、预览、筛选、分页查看项目
  - 支持按项目销售、反馈标签、月份、支撑人员、工单标签筛选
  - 支持批量勾选和批量删除
  - 支持 Excel 导入、导出
- 人员绩效
  - 支持按年、月、周维度统计绩效
  - 支持查看个人每日项目任务和绩效明细
  - 支持一键生成个人绩效和整月绩效汇总
- 出勤管理
  - 根据项目安排自动生成出勤状态
  - 管理员可手动调整为空闲或调休
- 项目文件
  - 支持文件上传、下载、预览
  - 支持按关键字搜索文件内容
- 统计报表
  - 支持数据概览、图表统计、项目导出预览
  - 支持按当前数据导出 Excel
- 账号权限
  - `admin` 系统管理员
  - `editor` 普通用户

## 运行环境

- PHP 7.4 及以上
- MySQL 5.7 及以上
- Apache / Nginx / phpStudy

## 安装部署

项目部署后请先访问：

```text
/install.php
```

安装页支持：

- 测试数据库连接
- 检查数据库是否存在
- 按需自动创建数据库
- 初始化表结构
- 初始化系统基础配置
- 创建首个系统管理员账号
- 自动写入 `config/app.php`

说明：

- 安装流程不会写入测试项目数据
- 首次安装由你自行配置系统管理员账号和密码
- 安装完成后再访问 `index.php`

## 本地运行

1. 将项目放入站点目录，例如 `F:\phpstudy_pro\WWW\pre`
2. Web 根目录指向项目根目录
3. 访问 `install.php` 完成数据库初始化
4. 安装完成后访问 `index.php`

## Excel 说明

### 项目导入

- 支持 `.xlsx`、`.csv`
- 导入字段已和当前导出模板对齐
- 支持将 Excel 内容直接导入平台

### 项目导出

- 支持导出项目计划表
- 已包含反馈标签列
- 已将“销售评价”导出到对应字段

### 绩效导出

- 支持一键生成整月绩效汇总
- 支持一键生成个人绩效明细

## 权限说明

- 系统管理员：可管理项目、人员绩效、出勤、文件、报表、系统设置和账号
- 普通用户：可新增项目、查看项目、查看出勤，不可进入人员绩效和报表管理

## 账号管理
