# 技术支撑事业部项目管理平台 V2.0

基于 `PHP + MySQL` 的轻量级项目管理系统，面向技术支撑事业部的项目录入、执行跟踪、人员绩效、出勤台账、项目文件与统计报表场景。

当前版本重点完成了项目数据结构升级、Excel 模板对齐、绩效自动折算、安装初始化流程、移动端与后台界面优化。

## 核心功能

### 1. 项目管理

- 支持新增、编辑、预览、删除项目
- 支持项目销售、反馈标签、月份、支撑人员、工单标签等组合筛选
- 支持按支撑人员排序、分页查看、批量选择
- 支持 Excel 导入项目数据
- 支持 Excel 导出项目数据
- 支持按模板字段管理项目属性

当前项目字段已覆盖：

- 项目类型
- 项目区域
- 项目名称
- 项目重要程度
- 项目销售
- 支撑事业部
- 跨部门协调
- 支撑岗位
- 支撑人员
- 开始时间
- 结束时间
- 项目工时
- 工作任务
- 标签
- 技术完成回执单
- 反馈标签
- 完成评价

### 2. 人员绩效

- 支持按 `年 / 月 / 周` 查看绩效
- 支持查看技术人员每日项目任务明细
- 支持管理员按“单日单项目”调整绩效
- 支持一键生成整月绩效汇总表
- 支持一键生成个人绩效表

绩效计算规则已升级为自动关联项目时间：

- 工作时段按 `09:00-12:00`、`13:00-18:00` 计算
- 满 `8` 小时计 `1` 绩效
- 不满 `8` 小时按占比折算
- 跨天项目会自动拆分到每日绩效
- 同一天多个项目可分别记录不同绩效

### 3. 出勤管理

- 根据项目安排自动生成出勤台账
- 管理员可手动将技术人员状态调整为“调休”
- 普通用户可查看出勤，不可修改

### 4. 项目文件

- 支持上传、下载、管理项目文件
- 支持按关键字搜索已上传文件

### 5. 统计报表

- 支持数据概览
- 支持项目导出预览
- 支持按当前数据导出 Excel

### 6. 账号与权限

系统已简化为两类角色：

- `admin`：系统管理员
- `editor`：普通用户

权限说明：

- 系统管理员：可管理项目、绩效、出勤、报表、文件与账号
- 普通用户：可新增项目、查看项目、查看出勤，不可管理人员绩效与系统账号

## 安装部署

项目部署后，请先访问：

```text
/install.php
```

安装页支持：

- 测试数据库连接
- 检查数据库是否存在
- 按需自动创建数据库
- 初始化数据表结构
- 写入系统配置
- 创建首个系统管理员账号

说明：

- 新安装流程不会写入测试项目数据
- 首次安装由你自行设置系统管理员账号和密码
- 安装完成后再访问 `index.php`

## 本地运行

1. 将项目放入站点目录，例如 `F:\phpstudy_pro\WWW\pre`
2. 确保 PHP 与 MySQL 环境可用
3. 打开浏览器访问 `install.php`
4. 完成安装后访问 `index.php`

## Excel 导入导出说明

### 项目导入

系统已按当前业务模板对齐，支持从 Excel 直接导入平台。

支持格式：

- `.xlsx`
- `.csv`

当前导入导出字段顺序已对齐以下模板列：

1. 项目类型
2. 序号
3. 项目区域
4. 项目名称
5. 项目重要程度
6. 项目销售
7. 支撑事业部
8. 跨部门协调
9. 支撑岗位
10. 支撑人员
11. 开始时间
12. 结束时间
13. 项目工时
14. 工作任务【简述】
15. 标签
16. 技术完成回执单
17. 完成评价【销售反馈简述】

### 项目导出

- 导出字段已与当前导入模板保持一致

### 绩效导出

- 支持一键生成整月绩效汇总表
- 支持一键生成个人绩效明细表
- 支持按项目拆分单日绩效记录
